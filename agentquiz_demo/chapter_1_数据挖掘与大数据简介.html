第一章：数据挖掘与大数据简介

大数据4V特征 · 数据挖掘任务 · KDD知识发现过程

核心知识点总结

一、基本概念

  • 大数据：无法在 一定时间内用常规软件工具 对其内容进行 抓取、管理、处理 的数据集合。需要更新处理模式才能适应海量、高增长率和多样化的信息资产。
  • 数据挖掘 (DM)：从大量数据中挖掘那些 令人感兴趣的、有用的、隐含的、先前未知的、可能有用的模式或知识。核心是"从大量数据中挖掘令人感兴趣的模式或知识"。

二、大数据的4V特征

特征英文含义
数据量大Volume数据规模巨大，TB/PB级别
多样性Variety数据类型多样，结构化、半结构化、非结构化数据
价值密度低Value大量数据中有价值信息占比低
速度快Velocity数据产生和处理速度快，实时性要求高
记忆口诀（4V）：
✦ 大(Volume) — 数据量大
✦ 多(Variety) — 类型多样
✦ 低(Value) — 价值密度低
✦ 快(Velocity) — 处理速度快

三、数据挖掘的四大主要任务

  • ① 关联（规则）分析 — 发现数据项之间有趣的关联关系（如"购买啤酒的人往往也买尿布"）
  • ② 聚类分析 — 将数据归为不同的类/簇，使得同类数据相似度最高、不同类差异最大
  • ③ 分类、预测与回归 — 找出描述和区分数据类的模型，预测未知对象的类标签
  • ④ 离群点分析 — 分析孤立而原本会被当作噪音丢弃的数据，如欺诈检测、入侵检测
关键区分：分类 vs 聚类 — 分类是有监督的（已知类标签），聚类是无监督的（无类标签，自动形成类别）。

四、KDD 知识发现过程（7步）

步骤名称说明
1数据清理消除噪声和删除不一致数据
2数据集成多种数据源可以组合在一起
3数据选择从数据库中提取与分析任务相关的数据
4数据挖掘 ⭐核心步骤，使用智能方法提取数据模式
5模式评估根据兴趣度度量，识别真正有趣的模式
6知识表示使用可视化和知识表示技术，向用户提供挖掘的知识
注意：KDD过程 = 数据清理 → 数据集成 → 数据选择 → 数据变换 → 数据挖掘（核心） → 模式评估 → 知识表示。共7步，其中数据挖掘是核心步骤！

五、数据挖掘的挑战

  • 数据容量 — 数据规模巨大，需要可伸缩的算法
  • 数据实时性 — 数据流不断到达，要求实时处理
  • 数据多样性 — 不同格式、不同来源的数据需要统一处理
  • 数据不确定性 — 缺失值、噪声数据、不一致数据

互动自测

—— 选择题 ——

Q1大数据的4V特征不包括以下哪一项？（ ）单选
A. Volume（数据量大）
B. Validity（有效性）
C. Variety（多样性）
D. Value（价值密度低）
✓ B
解析：大数据的4V特征是 Volume（数据量大）、Variety（多样性）、Value（价值密度低）、Velocity（速度快）。Validity（有效性）不属于4V特征。
Q2数据挖掘是从（ ）数据中挖掘令人感兴趣的、有用的、隐含的模式或知识。单选
A. 少量
B. 结构化
C. 大量
D. 干净的
✓ C
解析：数据挖掘的定义核心是"从大量的数据中挖掘……模式或知识"。关键词是"大量"。
Q3以下哪项不属于数据挖掘的四大主要任务？（ ）单选
A. 关联规则分析
B. 聚类分析
C. 分类与预测
D. 数据可视化
✓ D
解析：数据挖掘的四大主要任务是：①关联规则分析 ②聚类分析 ③分类/预测/回归 ④离群点分析。数据可视化是KDD过程中"知识表示"环节的手段，不是四大任务之一。
Q4在 KDD 知识发现过程中，核心步骤是（ ）。单选
A. 数据挖掘
B. 数据清理
C. 数据集成
D. 知识表示
✓ A
解析：KDD过程包括7步，其中数据挖掘是核心步骤，使用智能方法提取数据模式。其他步骤（清理、集成、选择、变换）是预处理环节，模式评估和知识表示是后处理环节。
Q5KDD 知识发现过程共包含几个步骤？（ ）单选
A. 5步
B. 6步
C. 7步
D. 8步
✓ C
解析：KDD过程共7步，依次为：数据清理 → 数据集成 → 数据选择 → 数据变换 → 数据挖掘 → 模式评估 → 知识表示。
Q6以下关于分类与聚类的描述，正确的是（ ）。单选
A. 分类和聚类都是无监督学习方法
B. 分类是无监督学习，聚类是有监督学习
C. 分类和聚类都是有监督学习方法
D. 分类是有监督学习，聚类是无监督学习
✓ D
解析：分类是有监督学习，需要已知类标签的训练数据；聚类是无监督学习，不需要类标签，自动将数据分组。选项A、B、C的监督关系均描述错误。
Q7离群点分析在以下哪个应用场景中最有价值？（ ）单选
A. 商品推荐系统
B. 欺诈检测
C. 天气预报
D. 图像分类
✓ B
解析：离群点分析的核心应用是欺诈检测和入侵检测。异常数据（如异常交易、异常登录）往往意味着欺诈或入侵行为。商品推荐适合关联规则，天气预报适合回归，图像分类适合分类算法。
Q8数据挖掘面临的挑战中，"Different format of data, different sources"对应（ ）。单选
A. 数据多样性 (Variety)
B. 数据容量 (Volume)
C. 数据实时性 (Velocity)
D. 数据不确定性
✓ A
解析："不同格式、不同来源的数据"对应4V中的Variety（多样性）。B对应数据规模大，C对应处理速度快，D对应缺失值和噪声数据。
Q9KDD过程中，消除噪声和删除不一致数据的步骤称为（ ）。单选
A. 数据集成
B. 数据变换
C. 数据清理
D. 数据选择
✓ C
解析：数据清理的任务就是消除噪声和删除不一致数据。数据集成是组合多数据源，数据变换是统一数据格式，数据选择是提取与任务相关的数据。
Q10以下哪个不是 KDD 过程中的步骤？（ ）单选
A. 数据清理
B. 数据采集
C. 数据变换
D. 模式评估
✓ B
解析：KDD的7步不包括"数据采集"。KDD假设数据已经存在于数据库中。7步为：数据清理 → 数据集成 → 数据选择 → 数据变换 → 数据挖掘 → 模式评估 → 知识表示。

数据挖掘
