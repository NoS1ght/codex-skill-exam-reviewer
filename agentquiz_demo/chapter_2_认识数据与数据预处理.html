第二章：认识数据与数据预处理

属性类型 · 统计描述 · 相似性度量 · 数据预处理

核心知识点总结

一、属性类型

大类子类特征示例主要用途
分类型
(定性)		标称型只在有限目标集中取值，值之间无序ID号、眼球颜色、邮政编码分类
序数型值之间有顺序关系，但差值无意义军阶、GPA、{tall,medium,short}
数值型
(定量)		区间差值有意义，但无绝对零点日历日期、摄氏温度回归
比率有绝对零点，比值有意义开氏温度、长度、计数
特殊：二元属性是标称型的一种，只有两个状态（0/1、是/否）。
✦ 对称二元：两个状态同等重要（如男/女）
✦ 非对称二元：两个状态权重不同（如阳性/阴性）

二、数据的统计描述

类别度量公式/定义
中心趋势度量均值 (Mean)所有值之和 / 值个数
中位数 (Median)排序后中间位置的值
众数 (Mode)出现频率最高的值
中列数 (Midrange)(max + min) / 2
数据散布度量极差 (Range)max - min
四分位数Q1 (25%), Q2/Median (50%), Q3 (75%)
四分位数极差 (IQR)IQR = Q3 - Q1
方差 / 标准差σ² = Σ(x-μ)²/N
经验关系（非对称单峰数据）：
均值 - 众数 ≈ 3 × (均值 - 中位数) 即 mean-mode ≈ 3(mean-median)
记忆口诀：五数概括
✦ [min, Q1, median, Q3, max]
✦ 用五个数描述一个数据集的分布概况

三、相似性 / 相异性度量

1. 标称属性

不匹配率：d(i, j) = (p - m) / p
其中 m = 取值匹配的变量数目，p = 变量总数。即：不同值数 / 总数。

2. 二元属性

对象j: 1对象j: 0
对象i: 1q（都是1）r（i为1，j为0）
对象i: 0s（i为0，j为1）t（都是0）
  • 对称二元（简单匹配）：d = (r+s) / (q+r+s+t)
  • 非对称二元（Jaccard）：d = (r+s) / (q+r+s) — 忽略 t（负匹配）

3. 数值属性 — 距离度量

距离公式说明
闵可夫斯基 (Minkowski)d = [Σ|xᵢ - yᵢ|ᵖ]^(1/p)通用形式，p=1为曼哈顿，p=2为欧氏
曼哈顿距离d = Σ|xᵢ - yᵢ|一维绝对值之和（城市街区距离）
欧氏距离d = √[Σ(xᵢ - yᵢ)²]二维开根号（直线距离）

4. 数据标准化方法

  • 最大-最小法（Min-Max）：v' = (v - min) / (max - min)，映射到 [0, 1]
  • Z-Score 标准化：v' = (v - μ) / σ，转为均值为0、标准差为1的分布

四、数据预处理的四个步骤

步骤任务方法
1. 数据清理处理缺失值、噪声数据填补缺失值、光滑噪声、识别/删除离群点
2. 数据集成合并多数据源、消除冗余相关分析（数值型）、卡方检验（标称型）
3. 数据归约减小数据量但保持完整性采样、PCA主成分分析、特征筛选（信息增益）
4. 数据变换统一数据格式最大-最小归一化、Z-Score标准化、离散化
信息增益公式：
Gain(A) = Info(D) - Info_A(D)
= 整体熵 - 按属性A划分后的条件熵
信息增益越大，说明该属性的分类能力越强（不确定性减少越多）。
记忆口诀：数据预处理顺序
✦ 清（清理）→ 集（集成）→ 归（归约）→ 变（变换）
✦ 谐音："请（清）集（集）归（归）变（变）" — 请集合归变！

互动自测

—— 选择题 ——

Q1以下哪种属性属于标称型属性？（ ）单选
A. 温度（摄氏度）
B. 邮政编码
C. GPA成绩
D. 身高
✓ B
解析：邮政编码是标称型属性，取值只在有限集合中且无序。温度是区间数值型，GPA是序数型，身高是比率数值型。
Q2标称型目标变量主要用于（ ），数值型目标变量主要用于（ ）。单选
A. 回归；分类
B. 聚类；关联
C. 分类；回归
D. 关联；聚类
✓ C
解析：标称型目标变量只在有限集合中取值（如真/假），用于分类；数值型目标变量从无限集合中取值（如0.555），用于回归。
Q3以下哪个不是中心趋势度量？（ ）单选
A. 均值
B. 中位数
C. 众数
D. 极差
✓ D
解析：极差属于数据散布度量，反映数据离散程度。均值、中位数、众数、中列数属于中心趋势度量，反映数据中心位置。
Q4对于非对称的单峰数据，均值、中位数、众数之间的关系为（ ）。单选
A. mean - mode ≈ 3(mean - median)
B. mean = median = mode
C. median - mode ≈ 2(mean - mode)
D. mode - median ≈ mean + median
✓ A
解析：经验公式 均值-众数 ≈ 3×(均值-中位数)，即 mean-mode ≈ 3(mean-median)。B是正态分布的情况，D无意义。
Q5四分位数极差 IQR 的计算公式为（ ）。单选
A. Q2 - Q1
B. Q3 - Q2
C. Q3 - Q1
D. max - min
✓ C
解析：IQR = Q3 - Q1，即第75百分位与第25百分位之差。D选项 max-min 是极差（Range），不同概念。
Q6计算标称属性数据的相似性时，采用的方法是（ ）。单选
A. 欧氏距离
B. 不匹配率 d = (p-m)/p
C. 曼哈顿距离
D. 余弦相似度
✓ B
解析：标称属性是类别变量，不能直接计算数值距离，只能用不匹配率：不同值的数量/总变量数。欧氏和曼哈顿距离用于数值属性。
Q7对于非对称二元属性，计算相异性时一般（ ）。单选
A. 忽略 t（负匹配），d = (r+s)/(q+r+s)
B. 包含 t，d = (r+s)/(q+r+s+t)
C. 只计算 q/(q+r+s)
D. 使用欧氏距离
✓ A
解析：非对称二元属性中两个状态的权重不同（如阳性/阴性），因此忽略两个都为0（t）的情况，使用Jaccard系数：d = (r+s)/(q+r+s)。B是简单匹配，用于对称二元。
Q8以下哪种数据标准化方法将数据映射到均值为0、标准差为1的分布？（ ）单选
A. 最大-最小法 (Min-Max)
B. 小数定标法
C. 等宽离散化
D. Z-Score 标准化
✓ D
解析：Z-Score 标准化公式为 v'=(v-μ)/σ，结果均值为0、标准差为1。Min-Max映射到[0,1]区间，不同方法效果不同。
Q9数据预处理中，使用卡方检验主要用于（ ）。单选
A. 数据清理
B. 数据变换
C. 数据集成中的冗余分析（标称数据）
D. 数据归约中的采样
✓ C
解析：数据集成时需要检测冗余属性。对于标称数据用卡方检验，对于数值数据用相关分析。清理用填补/光滑，变换用归一化，归约用PCA/采样。
Q10五数概括包含的五个数值是（ ）。单选
A. 均值、中位数、众数、极差、方差
B. min, Q1, median, Q3, max
C. Q1, Q2, Q3, IQR, Range
D. min, mean, median, mode, max
✓ B
解析：五数概括 = [min, Q1, median, Q3, max]，即最小值、第一四分位数、中位数、第三四分位数、最大值。这五个数可以通过盒图直观展示。

—— 填空题 ——

FIB_1属性类型分为两大类别：分类型（包括标称型和序数型）和__________（包括区间和比率）。填空
数值型
解析：分类型用于分类任务，数值型用于回归分析。区间属性差值有意义但无绝对零（如摄氏温度），比率属性有绝对零（如开氏温度、长度）。
FIB_2闵可夫斯基距离中，当 p=1 时为__________距离，当 p=2 时为欧氏距离。填空
曼哈顿
解析：p=1：曼哈顿距离 = Σ|xᵢ-yᵢ|（绝对值之和）；p=2：欧氏距离 = √Σ(xᵢ-yᵢ)²。p→∞时为切比雪夫距离。
FIB_3数据预处理的四个步骤依次是：数据清理 → 数据集成 → __________ → 数据变换。填空
数据归约
解析：顺序：清→集→归→变。数据归约通过采样、PCA、特征筛选等方式减少数据量但保持完整性。
FIB_4中列数的计算公式为 __________。填空
(max + min) / 2
解析：中列数 = (最大值 + 最小值) / 2，是中心趋势度量的一种，但容易受极端值影响。
FIB_5信息增益 = 整体熵 - __________。信息增益越大，说明该属性的分类能力越强。填空
条件熵（按属性划分后的熵）
解析：Gain(A) = Info(D) - Info_A(D)。条件熵越小，说明在该条件下不确定性越低，信息增益越大。
