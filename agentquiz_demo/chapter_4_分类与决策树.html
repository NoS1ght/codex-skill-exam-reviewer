第四章：分类与决策树

决策树构造 · ID3/C4.5/CART · 信息增益计算 · 属性选择度量

核心知识点总结

一、分类基本概念

  • 分类：有监督学习，根据训练数据建立模型，预测未知对象的类标签。
  • 分类是一个两步过程：①模型构建（训练阶段）→ ②模型使用（预测阶段）

二、决策树构造流程

  1. 树以单个节点（根节点）开始，代表所有训练样本
  2. 如果样本都在同一个类，则该节点成为树叶，用该类标记
  3. 否则，选择最有分类能力的属性作为当前节点的分裂属性
  4. 根据该属性取值的不同，将数据集分为若干子集，每个取值形成一个分枝
  5. 对每个子集递归重复上述步骤
递归停止条件（满足其一即停止）：
① 给定节点的所有样本属于同一类
② 没有剩余属性可以用来进一步划分样本
③ 某一分枝没有满足该分类的样本，则以样本的多数类创建树叶
④ 一个属性出现在一个节点后，不必在其任何后代中再考虑

三、属性选择度量（三种经典算法）

算法度量公式特点
ID3信息增益 GainGain(A) = Info(D) - Info_A(D)选最高信息增益的属性；偏向多值属性
C4.5增益率 GainRatioGainRatio = Gain(A) / SplitInfo(A)规范化信息增益，克服偏向多值属性的问题
CARTGini 指数Gini = 1 - Σpᵢ²二叉树；选最小Gini的属性

四、信息增益详解（ID3，必考）

步骤：
① 计算整体信息熵：Info(D) = -Σ pᵢ × log₂(pᵢ)
其中 pᵢ 为 D 中属于类 Cᵢ 的比例

② 按属性 A 划分后的条件熵：Info_A(D) = Σ (|Dⱼ|/|D|) × Info(Dⱼ)
其中 Dⱼ 为 A 取值 vⱼ 的子集

③ 信息增益：Gain(A) = Info(D) - Info_A(D)

④ 选择：信息增益最大的属性作为分裂属性
经典计算例题（以 age 为例）：
数据集：14个样本，9个yes/5个no
① 整体熵 Info(D) = -(9/14)log₂(9/14) - (5/14)log₂(5/14) ≈ 0.940
② 按 age 划分：youth(5人:2yes+3no), middle_aged(4人:4yes+0no), senior(5人:3yes+2no)
③ Info_age(D) = (5/14)×0.971 + (4/14)×0 + (5/14)×0.971 ≈ 0.694
④ Gain(age) = 0.940 - 0.694 = 0.246

五、增益率（C4.5）

  • SplitInfo(A) = -Σ (|Dⱼ|/|D|) × log₂(|Dⱼ|/|D|) — 分裂信息（属性A自身的熵）
  • GainRatio(A) = Gain(A) / SplitInfo(A)
  • 解决 ID3 偏向多值属性（如 ID号）的问题

六、Gini 指数（CART）

Gini(D) = 1 - Σpᵢ²
Gini 指数越小，数据集的"纯净度"越高。选择使得划分后Gini指数最小的属性。
记忆口诀：三种分裂算法
✦ ID3 — 信息增益，越大越好
✦ C4.5 — 增益率，规范化，惩罚多值
✦ CART — Gini指数，越小越好，二叉树

—— 选择题 ——

Q1决策树构造中选择"最有分类能力的属性"时，ID3 算法使用的度量是（ ）。单选
A. Gini 指数
B. 信息增益
C. 增益率
D. 支持度
✓ B
解析：ID3 用信息增益，C4.5用增益率，CART用Gini指数。支持度是关联规则的度量。
Q2信息增益的计算公式为（ ）。单选
A. Gain = Info(D) + Info_A(D)
B. Gain = Info_A(D) - Info(D)
C. Gain = Info(D) - Info_A(D)
D. Gain = Info(D) / Info_A(D)
✓ C
解析：信息增益 = 整体熵 - 条件熵 = Info(D) - Info_A(D)。信息增益越大，说明该属性划分后不确定性减少越多，分类能力越强。
Q3C4.5 算法相对于 ID3 的主要改进是（ ）。单选
A. 使用增益率克服信息增益偏向多值属性的问题
B. 支持连续属性
C. 支持回归任务
D. 使用集成学习方法
✓ A
解析：C4.5 引入增益率来规范化信息增益。ID3 偏向多值属性（极端如ID号每个值一个样本则Info=0，增益最大），C4.5通过除以SplitInfo来惩罚多值。
Q4Gini 指数的取值范围和含义是（ ）。单选
A. [0,1]，越大越纯净
B. [-1,1]，绝对值越大越纯净
C. [0,∞)，越大越纯净
D. [0,0.5]或[0,1]，越小越纯净
✓ D
解析：Gini 指数 = 1 - Σpᵢ²，范围 [0, 1-1/n]（n为类别数）。Gini 越小表示数据集越纯净（当所有样本属同一类时 Gini=0）。CART 选择划分后Gini最小的属性。
Q5决策树构造的递归停止条件不包括（ ）。单选
A. 给定节点的所有样本属于同一类
B. 信息增益大于阈值
C. 没有剩余属性可用来进一步划分
D. 某一分枝没有满足该分支分类的样本
✓ B
解析：停止条件为：①样本同属一类 ②无剩余属性 ③空分枝（用多数类标记）。"信息增益大于阈值"是有的，但阈值是人为设定且不属于三种基本停止条件。
Q6ID3 算法的一个极端缺陷是：如果某个属性（如ID号）每个取值仅对应一个样本，则信息增益（ ）。单选
A. 为0
B. 为负
C. 达到最大（被优先选择）
D. 无法计算
✓ C
解析：ID3的多值偏向问题：每个样本单独一类时条件熵为0，信息增益达到最大，ID3会优先选它为分裂属性。但ID号没有泛化能力。C4.5用增益率解决此问题。
Q7在决策树信息增益计算中，`Info_A(D)` 表示（ ）。单选
A. 按属性 A 划分后的条件信息熵
B. 属性 A 自身的信息熵
C. 数据集的整体信息熵
D. 属性 A 的信息增益
✓ A
解析：Info_A(D) = Σ(|Dⱼ|/|D|)×Info(Dⱼ)，即按属性A划分后各子集熵的加权平均，也就是条件熵。整体熵是Info(D)，属性A自身熵是SplitInfo(A)。
Q8CART 算法生成的决策树是（ ）。单选
A. 多叉树
B. 三叉树
C. N叉树（与属性取值数相同）
D. 二叉树
✓ D
解析：CART（分类与回归树）生成的是二叉树，每次分裂只产生两个分支。ID3和C4.5生成多叉树。CART既可用于分类也可用于回归。
Q9如果数据集有14个样本（9个yes/5个no），则整体信息熵 Info(D) 最接近（ ）。单选
A. 0.5
B. 0.94
C. 1.0
D. 1.5
✓ B
解析：Info(D) = -(9/14)log₂(9/14)-(5/14)log₂(5/14) ≈ -(0.643)×(-0.637)-(0.357)×(-1.485) ≈ 0.409+0.530 = 0.939。分布略偏但接近0.94。
Q10增益率的计算公式中，SplitInfo(A) 的作用是（ ）。单选
A. 增大信息增益的值
B. 提高计算精度
C. 对多值属性进行惩罚（规范化）
D. 加快计算速度
✓ C
解析：SplitInfo(A)是属性A自身的熵，取值越多的属性SplitInfo越大。GainRatio=Gain/SplitInfo，除以较大的SplitInfo实际上惩罚了多值属性，从而克服ID3的偏向问题。

—— 填空题 ——

FIB_1决策树的三种属性选择度量分别是：ID3 使用信息增益，C4.5 使用__________，CART 使用 Gini 指数。填空
增益率（Gain Ratio）
解析：GainRatio(A) = Gain(A) / SplitInfo(A)。ID3→信息增益，C4.5→增益率，CART→Gini指数。
FIB_2分类是__________学习，需要已知类标签的训练数据；而聚类是无监督学习。填空
有监督（监督）
解析：分类需要带标签的训练数据，是典型的有监督学习。聚类无需标签，是无监督学习。这是两者的本质区别。
FIB_3信息熵公式为 Info(D) = -Σ__________ × log₂(pᵢ)，其中 pᵢ 是类别 i 的比例。填空
pᵢ
解析：Info(D) = -Σ pᵢ·log₂(pᵢ)。信息熵度量数据的混乱程度/不确定性。均匀分布时最大，完全纯净时最小（=0）。
FIB_4决策树递归停止的三个基本条件是：样本同属一类、__________、某一分枝没有满足该分类的样本。填空
没有剩余属性可以用来进一步划分
解析：三个停止条件：①样本同属一类 ②无剩余属性 ③空分枝（用多数类标记）。实际中也可设置最大深度、最小样本数等。
FIB_5Gini 指数公式为 Gini(D) = 1 - Σ__________。Gini 越小，数据集越纯净。填空
pᵢ²
解析：Gini = 1 - Σpᵢ²。当所有样本同一类时 p=1, Gini=0；均匀分布时Gini最大。CART选择划分后Gini加权和最小的属性。
